# 智糖小助手管理后台

## 🎉 功能特色

这是一个完整的、可交互的管理后台系统，包含以下功能：

### 👥 用户管理
- ✅ 用户列表查看（支持搜索和状态筛选）
- ✅ 创建新用户
- ✅ 编辑用户信息
- ✅ 启用/禁用用户状态
- ✅ 删除用户

### ❓ FAQ管理
- ✅ FAQ列表查看（支持分类筛选）
- ✅ 创建FAQ（问题+答案）
- ✅ 编辑FAQ内容
- ✅ 查看FAQ详情
- ✅ 批量删除FAQ

### 🏷️ 标签管理
- ✅ 查看标签定义
- ✅ 查询用户标签
- ✅ 批量设置标签
- ✅ 清空用户标签

### 💬 提示词管理
- ✅ 提示词模板列表
- ✅ 创建新模板
- ✅ 编辑模板
- ✅ 删除模板

### 📝 消息记录
- ✅ 消息历史查询
- ✅ 支持用户筛选
- ✅ 日期范围筛选
- ✅ 会话详情查看

### 📊 知识问答
- ✅ FAQ统计信息
- ✅ 知识库来源分析

## 🚀 快速开始

### 方法1：使用演示脚本（推荐）
```bash
cd admin-backend
./demo.sh
```

### 方法2：手动启动
```bash
cd admin-backend
python3 start_admin_server.py
```

### 访问地址
- **登录页面**: http://127.0.0.1:8901/admin/login
- **演示账号**: `admin` / `admin123`

## 🎨 界面特色

- **现代化设计**: 使用 Tailwind CSS 和渐变色
- **响应式布局**: 支持桌面和移动设备
- **丰富的交互**: 模态框、消息提示、加载动画
- **直观的导航**: 侧边栏导航，清晰的页面结构
- **实时反馈**: 操作结果即时提示

## 🛠️ 技术栈

- **前端**: HTML5, CSS3, JavaScript (ES6+)
- **样式**: Tailwind CSS
- **图标**: Font Awesome 6
- **后端**: Python Flask
- **数据库**: MySQL

## 📋 使用说明

### 1. 登录系统
- 使用演示账号 `admin` / `admin123` 登录
- 登录后会自动跳转到主界面

### 2. 导航操作
- 点击左侧导航栏切换不同功能模块
- 每个模块都有独立的搜索和筛选功能

### 3. 数据操作
- **创建**: 点击"创建"按钮添加新数据
- **编辑**: 点击"编辑"按钮修改现有数据
- **删除**: 点击"删除"按钮移除数据
- **搜索**: 使用搜索框快速找到数据

### 4. 退出登录
- 点击右上角的"退出"按钮安全退出

## 🎯 演示亮点

1. **完整的CRUD操作**: 创建、读取、更新、删除
2. **实时数据展示**: 表格、卡片等丰富展示形式
3. **交互反馈**: 加载状态、成功/失败提示
4. **数据验证**: 前端表单验证和后端API验证
5. **分页展示**: 大数据量分页显示
6. **搜索筛选**: 多维度数据筛选
